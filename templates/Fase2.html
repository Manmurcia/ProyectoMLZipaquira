Fase 2: Ingeniería de Datos - Proyecto CRISP-ML

Proyecto

Clasificación de Riesgo de Enfermedades Respiratorias en Adultos Mayores de Zipaquirá

Esta fase corresponde al procesamiento y transformación de los datos utilizados para entrenar el modelo de clasificación. Se utilizó un conjunto de datos sintético generado con fines de prueba y desarrollo.

1. Generación del Dataset (Data Seed)

Se creó un archivo llamado 'dataseed_respiratorio.csv' con 200 registros simulando adultos mayores de Zipaquirá. Las variables incluidas representan factores clínicos, ambientales y sociodemográficos relevantes para evaluar el riesgo respiratorio.

2. Variables Utilizadas

  • edad: Edad del adulto mayor (60 a 90 años)
  • sexo: Masculino o Femenino
  • comorbilidades: Presencia de enfermedades como Hipertensión o Diabetes
  • uso_leña: Uso de leña como combustible (0 = No, 1 = Sí)
  • tipo_vivienda: Condiciones de vivienda: Cerrada, Ventilada, Hacinada
  • pm25: Contaminación ambiental en µg/m³
  • consultas_respiratorias: Número de consultas médicas por afecciones respiratorias en el último año
  • estrato: Nivel socioeconómico (1 a 3)
  • actividad_fisica: Realiza actividad física semanalmente (0 = No, 1 = Sí)
  • riesgo_respiratorio: Etiqueta objetivo: Bajo, Medio o Alto

3. Limpieza y Transformación de los Datos

Se validó que no existieran valores nulos ni inconsistencias. Se transformaron variables categóricas con codificación adecuada y se normalizaron las variables numéricas para garantizar compatibilidad con los algoritmos de aprendizaje automático.

4. Resultado

El archivo final 'data_ready.csv' está preparado para su uso en la fase de modelado. El proceso completo está documentado en el script de preprocesamiento en Jupyter Notebook.

Esta fase fue asistida por ChatGPT (OpenAI) como apoyo en redacción y generación del dataset sintético.
